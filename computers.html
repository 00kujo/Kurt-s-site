Some examples of Computer Applications.

Spreadsheets are computer programs that allows the entry, calculation, and storage of data in columns and rows Examples include: LANPAR (1969), VisiCalc (1979), Microsoft Excel (1987)

Computer-aided design programs allow the user to manipulate geometry for the purpose of creation, modification, analysis for optimization of a design in either to 2-D or 3-D and are frequently used by engineers and architects. Examples include: AutoCAD (1982), SolidWorks (1995), SketchUp (2000)

A Web browser Is an application that allows the user to access the World Wide Web or a local website. Examples include: Mosaic (1993), Internet Explorer (1995), Google Chrome (2008)
