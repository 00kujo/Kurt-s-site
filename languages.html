Some examples of Progaming tools.

A Text editor is software that is designed for the programmer to edit the Programming language in playing text. Examples include: Windows Notepad (1983), TextEdit (1996), Visual Studio Code (2015)

Machine code is a computer program that is written in machine language that directly interacts with the computers central processing unit (CPU)

Assembly languages are any low level programming languages with instruction sets designed to communicate with the CPU. It requires an assembler to be executed into machine code
